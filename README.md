# 项目简介

## 核心功能
- 用户浏览导航条，包括首页、闲置物品、校园论坛、校园通知等模块
- 个人中心允许用户更新个人信息，包括密码修改、闲置物品管理、发布和收藏的编辑
- 管理员功能，包括首页管理、用户管理、物品分类、举报记录、论坛分类、系统管理以及个人中心等

## 适用场景
- 高校师生交流信息，发布和查看校园资讯
- 学生二手物品交易，促进资源共享
- 教师和学生交流学术话题，讨论校园生活

## 技术特点
- 架构：B/S架构、MVC模式、Web应用框架
- 开发环境：使用IDEA、JDK1.8、Mysql数据库、Maven依赖管理、Navcat数据库管理工具
- 技术栈：Java语言、SpringBoot框架、Vue前端框架
